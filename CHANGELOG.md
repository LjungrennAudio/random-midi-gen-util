# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by “Keep a Changelog”, and this project follows semantic versioning where practical.

## [0.0.1] - 2025-12-21
### Added
- Seeded stochastic MIDI generator (SMF format 0, single track).
- CLI options for seed, tempo (BPM), bars, PPQN, channel, program.
- `--root` parser supporting note names like `C4`, `F#3`, `Db2`.
- Scale selection via `--scale` (major, natural-minor, minor-pentatonic, major-pentatonic).
- Auto-generated timestamped output name when `--out` is omitted.
